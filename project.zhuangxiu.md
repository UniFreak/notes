# 监工
- 全部为指定产品

## 电
+ 电管内不超过 8 根, 同回路同管, 全部接地
+ 厨卫各一个断路器, 其他空气开关
+ 卫生间不走地板
+ 与暖气, 热水管分开 > 30cm

## 水
+ 冷热管分开 > 10cm, 冷热水不共槽, 固定住, 粗(6分管)细(4分管)结合
+ 冷热水管都要做保温层
+ 水压, 漏水
+ 覆盖水泥厚度: 冷 > 1cm, 热 > 1.5cm
+ 进水管加止水阀
+ 马桶有 p/s 存水湾
+ 试水压

## 泥瓦
- 泥浆
    + 泥沙 1:3 用于打底, 1:2 且筛沙用于面层; 搅拌机搅拌均匀
    + 浇水养护 14 天以上, 不要风扇加速干燥
- 瓷砖
    + 大瓷砖 (>60*60cm) 要上粘着剂, 30*30 以上用橡胶锤敲击防胀
    + 抛光砖要留缝 1~2mm
- 隔断用的砖墙
    + 用 1:2 水泥, 砂浆厚度 6~10mm
    + 用之前浇湿
    + 新旧墙之间打钢钉或植钢筋
    + 用机器定位仪量基准线, 拉白线定位置, 每隔一米拉两条线
    + 每天最多砌 1.2 米高
    + 等 3 周后再刮腻子
- 铝窗
    + 拆除时内墙四角都敲掉, 装窗时使用固定架(而非木片或报纸), 装窗后水泥填平, 压灌弹性发泡填缝剂, 水泥干透后再刮腻子
    + 若拆了外墙, 要做泄水破, 把拆掉的瓷砖补回来, 打中性玻璃胶
    + 铝窗应在瓦工退场前装好
- 退场前把所有孔洞, 管线沟槽, 门框窗框封平

## 厨卫
- 防水: 先壁再地, 水泥基地要干净平整, 防水浆料薄涂多层(各次之间间隔 2~4 小时), 3~7 天后贴砖
- 泄水坡每一米降 1cm, 粘砖前用乒乓球测试
- 贴砖前放样
- 24 小时候后填缝
- 卫生间吊顶要用防锈处理龙骨, 灯具或浴霸地方加吊杆, 设计维修孔
